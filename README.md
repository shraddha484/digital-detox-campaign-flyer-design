# Digital Detox Campaign Flyer Design

## 📌 Project Overview
This project showcases a creative flyer design created for a **Digital Detox Awareness Campaign**. The goal of the flyer is to encourage people to reduce screen time and reconnect with real-life activities such as reading, going outdoors, and practicing mindfulness.

The design communicates the message **“Disconnect to Reconnect”**, highlighting the benefits of unplugging from digital devices for better mental health, productivity, and sleep quality.

## 🎯 Objectives
- Promote awareness about excessive screen usage
- Encourage healthier lifestyle habits
- Deliver a clear and visually engaging message through print design

## 🖼 Design Concept
The flyer uses nature-inspired visuals where a **smartphone transforms into plants and natural elements**, symbolizing balance between technology and real life. Soft green and blue tones represent calmness, focus, and well-being.

## ✨ Key Design Elements
- Modern and clean layout
- Nature-themed illustration
- Minimal and readable typography
- Informative icons for quick understanding
- Clear call-to-action section

## 📋 Flyer Content Sections
1. **Headline:** Disconnect to Reconnect  
2. **Benefits of Unplugging**
   - Improve Mental Health
   - Boost Productivity
   - Better Sleep Quality
3. **Simple Steps**
   - Read a Book
   - Go Outside
   - Talk to People
   - Enjoy a Hobby
   - Practice Mindfulness
4. **Call to Action**
   - Join the 24-Hour Digital Detox Challenge

## 🛠 Tools Used
- Canva 
- Vector illustrations
- Icon-based visual storytelling

## 📁 Project Purpose
This project is created for **portfolio demonstration in graphic and print design**, showcasing the ability to communicate social awareness messages through visual design.
